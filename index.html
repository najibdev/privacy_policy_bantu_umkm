Kebijakan Privasi

Aplikasi Bantu UMKM

Selamat datang di Bantu UMKM. Kami menghormati privasi Anda dan berkomitmen untuk melindungi data pribadi yang Anda percayakan kepada kami. Kebijakan Privasi ini menjelaskan informasi apa yang kami kumpulkan, bagaimana kami menggunakannya, dan hak-hak Anda sebagai pengguna.

Dengan menggunakan aplikasi ini, Anda menyetujui praktik yang dijelaskan dalam kebijakan ini. Jika Anda tidak setuju, harap hentikan penggunaan aplikasi.

1. Data yang Anda Masukkan dan Penyimpanan Lokal

Semua data bisnis Anda disimpan sepenuhnya di perangkat Anda sendiri — tidak ada data transaksi, produk, pelanggan, atau keuangan yang dikirim ke server kami.

Aplikasi Bantu UMKM menggunakan database lokal (SQLite/Room) yang tersimpan di memori internal perangkat Anda. Data tersebut meliputi:

  • Data produk, bahan baku, dan layanan
  • Data transaksi penjualan dan pembelian
  • Data pemasok dan informasi bisnis lainnya
  • Laporan dan kalkulasi keuangan

Data ini sepenuhnya berada di bawah kendali Anda. Jika aplikasi dihapus (uninstall), seluruh data lokal akan ikut terhapus dari perangkat Anda.

2. Fitur Backup ke Google Drive (Opsional)

Aplikasi menyediakan fitur backup dan pemulihan data secara opsional menggunakan Google Drive. Fitur ini hanya aktif jika Anda memilih untuk menggunakannya.

Proses login Google

Untuk menggunakan fitur backup, Anda perlu masuk menggunakan akun Google Anda. Proses autentikasi ini ditangani langsung oleh layanan Google (Google Sign-In) — kami tidak pernah menyimpan kata sandi atau token akses Anda di server kami.

Data yang dikirim ke Google Drive

Saat Anda melakukan backup, salinan terenkripsi dari database lokal Anda akan diunggah ke folder privat aplikasi di Google Drive Anda sendiri. Data ini hanya dapat diakses oleh Anda melalui akun Google yang sama.

Penggunaan data dari Google Drive tunduk pada Kebijakan Privasi Google.

3. Firebase Analytics

Kami menggunakan Firebase Analytics (layanan Google) untuk memahami cara pengguna berinteraksi dengan aplikasi. Data yang dikumpulkan bersifat anonim dan agregat, meliputi:

  • Halaman atau fitur yang dikunjungi
  • Durasi sesi penggunaan aplikasi
  • Jenis perangkat, versi sistem operasi, dan bahasa
  • Negara/wilayah pengguna (berdasarkan IP, tidak disimpan)
  • Kejadian (event) tertentu dalam aplikasi yang kami definisikan

Data analytics tidak mengandung informasi bisnis yang Anda masukkan ke aplikasi. Firebase Analytics menggunakan ID perangkat anonim. Anda dapat menonaktifkan pengumpulan data analytics melalui pengaturan Google di perangkat Anda.

Informasi lebih lanjut: Privasi Firebase Google.

4. Firebase Crashlytics

Kami menggunakan Firebase Crashlytics untuk mendeteksi dan memperbaiki kesalahan (crash/error) pada aplikasi. Ketika aplikasi mengalami crash, laporan otomatis dikirim yang berisi:

  • Stack trace (jejak kesalahan teknis)
  • Jenis perangkat dan versi sistem operasi
  • Versi aplikasi yang digunakan
  • Waktu terjadinya crash
  • Status memori dan CPU perangkat saat crash

Laporan crash tidak mengandung data pribadi atau data bisnis yang Anda simpan dalam aplikasi. Laporan ini semata-mata digunakan untuk meningkatkan stabilitas aplikasi.

5. Firebase Remote Config

Kami menggunakan Firebase Remote Config untuk mengonfigurasi perilaku aplikasi dari jarak jauh tanpa perlu memperbarui versi aplikasi. Fitur ini digunakan untuk:

  • Mengaktifkan atau menonaktifkan fitur tertentu
  • Menyesuaikan tampilan atau konten aplikasi
  • Pengumuman pembaruan penting

Remote Config tidak mengumpulkan data pribadi Anda — ia hanya mengirimkan konfigurasi dari server kami ke aplikasi Anda.

6. Iklan (Google AdMob)

Aplikasi versi gratis menampilkan iklan melalui Google AdMob. Untuk menampilkan iklan yang relevan, AdMob dapat mengumpulkan dan menggunakan:

  • ID Iklan (Advertising ID) perangkat Anda
  • Informasi perangkat (model, OS, bahasa)
  • Lokasi perkiraan berdasarkan jaringan (bukan GPS)
  • Data interaksi dengan iklan (klik, tayangan)
  • Minat dan preferensi berdasarkan aktivitas di aplikasi Google lainnya

Anda dapat mengontrol iklan yang dipersonalisasi melalui pengaturan Privasi di perangkat Anda (Setelan > Google > Iklan). Untuk menonaktifkan iklan sepenuhnya, Anda dapat membeli fitur "Hapus Iklan" yang tersedia dalam aplikasi.

Informasi lebih lanjut tentang cara Google menggunakan data untuk iklan: Kebijakan Iklan Google.

7. Pembelian Dalam Aplikasi (Google Play Billing)

Aplikasi ini menawarkan pembelian dalam aplikasi (in-app purchase) melalui Google Play Billing, seperti fitur "Hapus Iklan". Semua transaksi pembayaran diproses sepenuhnya oleh Google Play — kami tidak menyimpan informasi kartu kredit atau detail pembayaran Anda. Data pembelian tunduk pada Kebijakan Privasi Google Payments.

8. Kamera dan Pemindai Barcode

Aplikasi meminta izin akses kamera untuk fitur pemindaian barcode produk menggunakan ML Kit Google. Kamera hanya aktif saat Anda secara eksplisit membuka fitur pemindai. Kami tidak merekam, menyimpan, atau mengunggah gambar atau video dari kamera Anda ke server manapun. Pemrosesan barcode dilakukan sepenuhnya di perangkat secara lokal (on-device).

9. Notifikasi

Dengan izin Anda, aplikasi dapat mengirimkan notifikasi lokal untuk:

  • Pengingat tugas dan aktivitas bisnis
  • Pemberitahuan backup otomatis selesai
  • Informasi pembaruan aplikasi

Notifikasi ini dihasilkan dan dikelola sepenuhnya di perangkat Anda. Anda dapat menonaktifkan notifikasi kapan saja melalui pengaturan perangkat.

10. Data yang Tidak Kami Kumpulkan

Kami secara tegas menyatakan bahwa kami tidak:

  • Mengumpulkan atau menyimpan data transaksi, produk, atau keuangan Anda di server kami
  • Menjual data Anda kepada pihak ketiga
  • Mengumpulkan nama, alamat email, nomor telepon, atau identitas pribadi Anda (kecuali yang diberikan sukarela untuk keperluan dukungan)
  • Melacak lokasi GPS Anda
  • Mengakses kontak, galeri foto, atau file lain di luar kebutuhan fitur yang Anda aktifkan

11. Berbagi Data dengan Pihak Ketiga

Kami tidak menjual data Anda. Data hanya dibagikan kepada layanan pihak ketiga berikut semata-mata untuk menjalankan fungsi aplikasi:

  • Google LLC — Firebase Analytics, Crashlytics, Remote Config, AdMob, Google Drive API, Google Sign-In, Google Play Billing, ML Kit

Semua layanan Google di atas tunduk pada Kebijakan Privasi Google.

12. Keamanan Data

Kami mengambil langkah-langkah teknis yang wajar untuk melindungi data Anda:

  • Data bisnis hanya tersimpan lokal di perangkat Anda dan dilindungi oleh sistem keamanan Android (sandboxing aplikasi)
  • Komunikasi dengan layanan Google menggunakan enkripsi HTTPS/TLS
  • File backup di Google Drive dilindungi oleh izin akses akun Google Anda

Meskipun demikian, tidak ada metode transmisi atau penyimpanan yang 100% aman. Kami mendorong Anda untuk menggunakan PIN/password/biometrik di perangkat Anda untuk lapisan perlindungan tambahan.

13. Privasi Anak-Anak

Aplikasi Bantu UMKM ditujukan untuk pengguna berusia 13 tahun ke atas (atau sesuai batas usia minimum di negara Anda). Kami tidak secara sengaja mengumpulkan data dari anak-anak di bawah usia tersebut. Jika Anda mengetahui bahwa anak di bawah umur telah memberikan data pribadi kepada kami, harap hubungi kami agar kami dapat mengambil tindakan yang sesuai.

14. Hak Anda sebagai Pengguna

Anda memiliki hak untuk:

  • Mengakses — Data bisnis Anda sepenuhnya tersedia dan dapat dilihat langsung di aplikasi
  • Menghapus — Menghapus data dari aplikasi kapan saja, atau uninstall aplikasi untuk menghapus semua data lokal
  • Memindahkan — Mengekspor data bisnis Anda melalui fitur ekspor yang tersedia
  • Membatasi — Menonaktifkan iklan yang dipersonalisasi atau Analytics melalui pengaturan Google di perangkat Anda
  • Mencabut akses backup — Mencabut izin akses Google Drive kapan saja melalui pengaturan akun Google Anda di myaccount.google.com/permissions

15. Retensi Data

Data lokal disimpan selama aplikasi terpasang di perangkat Anda. Data backup di Google Drive disimpan sesuai kebijakan penyimpanan Google Drive Anda sendiri — Anda dapat menghapusnya kapan saja langsung dari Google Drive. Data anonim yang dikumpulkan oleh Firebase disimpan sesuai kebijakan retensi Firebase (umumnya 2 bulan hingga 14 bulan tergantung jenis data).

16. Perubahan Kebijakan Privasi

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Perubahan signifikan akan diinformasikan melalui notifikasi di dalam aplikasi atau pembaruan versi aplikasi. Tanggal "Berlaku efektif" di bagian atas halaman ini akan selalu mencerminkan versi terbaru. Kami mendorong Anda untuk meninjau kebijakan ini secara berkala.

Penggunaan berkelanjutan terhadap aplikasi setelah perubahan kebijakan berlaku dianggap sebagai persetujuan Anda terhadap kebijakan yang diperbarui.
